Intro

This the cornerstone of T-Rex Solutions LLC's (T-REX) Centers of Excellence (COE) Technology Lab Projects (TLP).

Greenfield

The Account Vending Machine (AVM) is responsible for making the Landing Zone Hubs and Spokes and handing them off to the provisioner.

The provisioner, this application, is an AWS CDK (node js) app, will deploy the 4 Stacks after receiving them from the AVM.

This requires bootstrapping the Central Account via your laptop. v2.0.0 will feature an AWS Marketplace Entry

Brownfield

Directly provision an account you can login to with AdministratorAccess. This needs to be battle tested with many different environments; however, it does work in MANY environments.

Description

T-Rex SMILE™ enables faster, simpler, more secure, and cheaper cloud migrations by providing Secured, Managed Infrastructures, Landing Zones, and Environments (SMILE) via Infrastructure as Code (IaC) and Continuous Configuration Automation (CCA) that aligns to current best practices as a pre-built solution.
